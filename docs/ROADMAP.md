# 🗺️ Roadmap

## Q3 2026 (Current Quarter)

### 🔴 High Priority
- [x] Architecture design & documentation
- [x] Database schema (PostgreSQL)
- [x] Go microservices (Auth, Problem, Execution, Leaderboard, Hint)
- [x] React frontend with Monaco Editor
- [x] Docker sandbox code execution
- [x] Hint system (3 levels)
- [x] Leaderboard & Rating (ELO)
- [x] CI/CD pipeline (GitHub Actions)
- [x] Monitoring (Prometheus + Grafana)
- [x] Infrastructure as Code (Terraform)

### 🟡 Medium Priority (Next)
- [ ] **Multiple Programming Languages** — Python, JavaScript, Java, C++
- [ ] **Contest System** — Weekly/monthly coding contests
- [ ] **Discussion Forum** — Per-problem discussion threads
- [ ] **Code Review** — Peer code review feature
- [ ] **Subtitle/Problem Editor** — WYSIWYG editor for problem creation
- [ ] **Email Notifications** — Submission results, contest reminders

### 🟢 Low Priority (Future)
- [ ] **AI-Powered Hints** — GPT-based adaptive hints
- [ ] **Code Explanation** — AI explanation for solutions
- [ ] **Interview Mode** — Simulate technical interviews
- [ ] **Company-Specific Problems** — Google, Meta, Amazon tagged problems
- [ ] **Mobile App** — React Native for iOS/Android
- [ ] **Social Features** — Friends, groups, team contests
